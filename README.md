**가계부 백엔드 API**

**프로젝트 개요**

명확한 기능과 설명가능한 설계를 목표로 시작합니다

개인 사용자의 가계부 웹서비스로 수입, 지출을 기록하고 월별 카테고리 통계를 제공하는 것을 목표로 합니다. REST API 설계, 인증/인가, 데이터베이스 모델링, 공공데이터 API, 통계 집계 로직에 중점을 두고 구현합니다.

**기술 스택**

## Backend

- Java 17
- Spring Boot
- Spring MVC
- Spring Security (JWT 기반 인증)
- JPA (Hibernate)

## Database

- MySQL

## Frontend

- Thymeleaf (또는 React 기반 단순 UI)

## Infra Tool

- Gradle
- Swagger(OpenAPI)
- AWS EC2
